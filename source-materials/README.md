# Source Materials

Drop raw materials here. I'll build tooling to process them.

## Folders

- **emails/** — Weekly letters home (any format: .txt, .html, .eml, copy-paste docs)
- **messenger/** — Messenger conversation exports or screenshots
- **photos/** — All mission photos (subfolders by transfer/area if possible, otherwise just dump them)
- **companions/** — Companion list (names, dates, areas) — a text file, spreadsheet, whatever you have
- **transfers/** — Transfer history (areas, dates, companions per transfer)
- **other/** — Anything else: journal entries, maps, certificates, etc.
